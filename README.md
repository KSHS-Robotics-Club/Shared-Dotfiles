# Shared-Dotfiles
Some shared dotfiles designed to provide an easy to use out of the box Linux experience for less experienced members.

## Setup
To setup run 
```sh
git clone https://github.com/KSHS-Robotics-Club/Shared-Dotfiles.git ~/Dotfiles
cd ~/Dotfiles
chmod +x install.sh
sudo ./install.sh
```

## Features
- neovim
- tmux
